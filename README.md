# IP Geolocator 🗺️
Enter any IP address and see it plotted on an interactive map with full location details.
Uses the free ip-api.com service — no API key needed.

## Features
- Real-time geolocation lookup
- Interactive Leaflet map
- ISP, timezone, country info
- Batch lookup via CSV upload
- Mobile responsive

## Run
```bash
npm install && npm start
```
